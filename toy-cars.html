Toy Cars

This was my first game project with Unity. It is a physics based puzzle game with hotwheel-like mechanics. Baically, the player arranges different types of tracks to get the car form the start point to the finish point. I worked on the programming of the game while a friend worked on the game design.

Although the game is quite fleshed out; it has a level system, scoring, coins, track penetration detection, UI and more, we had to cancle the project because of the inconsistency of physics in Unity across platforms. A deterministic physics engine had to be made to overcome this issue, which was beyond what we had intended.
